# Website Plan By Phase

## Phase 0: Discovery And Assets

Goal:

Collect the business truth before designing.

Tasks:

- Confirm final company name, address, phone, email, domain, and social links.
- Confirm target industries and highest-value client type.
- Confirm exact services offered.
- Confirm metrics and claims.
- Collect real team/agent photos.
- Collect agent voice clips.
- Export crisp logo files from the `.ai` sources.
- Use navy/white with restrained logo red as the starting accent direction, then validate visually.
- Decide form destination email/provider.

Output:

- Approved content brief.
- Clean asset folder.
- Final Phase 1 scope.

## Phase 1: Static Modern Website

Goal:

Launch a polished, fast, responsive, SEO-ready business website.

Pages:

- Home
- Services
- About Us
- FAQ
- Contact Us

Core sections:

- Hero with clear CTA.
- Services overview.
- Industries served.
- Key workforce statistics.
- How it works/process.
- Testimonials/trust signals.
- About/team/agents.
- Book a Free Demo form.
- Footer.

Tech:

- Astro + Tailwind CSS.
- npm for dependencies and build scripts.
- Static content files.
- Cloudflare Pages + GitHub.
- Third-party form backend or simple Cloudflare handler.

Output:

- Live website on Cloudflare Pages.
- Custom domain connected.
- Basic SEO and analytics configured.
- Client handoff notes.

## Phase 2: CMS For Owner Editing

Goal:

Let the owner update selected content without asking a developer for every change.

Recommended:

- Sanity CMS with Astro.

Editable content:

- Services.
- FAQs.
- Testimonials.
- Agent profiles.
- Agent photos.
- Agent voice clips.
- Stats copy, if approved.
- Contact details.

Important:

- This is a separate paid scope.
- Do not add until the owner confirms they actually need it.

## Phase 3: Conversion And Trust Improvements

Goal:

Improve lead quality and trust after launch.

Possible additions:

- Case studies.
- Client logos.
- Industry-specific landing pages.
- Better testimonials with photos/company permission.
- Calendly or booking integration.
- CRM integration.
- A/B testing CTAs.
- More robust form routing.

## Phase 4: SEO And Maintenance

Goal:

Keep the site healthy and improve visibility over time.

Tasks:

- Search Console monitoring.
- Analytics review.
- Content updates.
- Performance checks.
- Broken link checks.
- Dependency updates.
- New pages for services/industries.
- Monthly backup/export of CMS content if CMS is added.

## Scope Guardrails

Phase 1 is not:

- A custom CMS.
- A full admin dashboard.
- A marketing automation platform.
- A CRM.
- A complex web app.

Phase 1 is:

- A high-quality official website that builds trust and converts visitors into demo requests.
