# Americall Phase 1

This folder contains the planning and operating documents for the Americall website redesign.

Start here:

- `AGENT_START_HERE.md`
- `progress.md`
- `project-context.md`
- `design-direction.md`
- `technical-plan.md`

The current Phase 1 direction is:

```txt
Astro + Tailwind CSS + npm + GitHub + Cloudflare Pages
```

The buildable website app should be created in `americall-site/`.

Firebase is not required for Phase 1. A CMS/admin system should be treated as Phase 2 after the owner confirms they need self-editing.
